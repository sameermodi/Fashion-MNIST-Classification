# Fashion-MNIST Classification using CNN

This project classifies Fashion-MNIST images using Deep Learning models.

## 🚀 Models Used
- ANN
- Basic CNN
- Deep CNN

## 🛠 Technologies
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## 📊 Dataset
Fashion-MNIST dataset containing:
- T-shirt
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle boot

## 🎯 Accuracy
- ANN: ~88%
- CNN: ~92%

## 📌 Features
- Image classification
- CNN visualization
- Prediction analysis
- Error analysis
